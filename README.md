# Danbooru Tags Translator Web App

![](./assets/screenshot.jpg)

## Backend server

Setup:

```
uv sync
```

Run backend server on Linux:

```
./scripts/server.sh
```

Run backend server on Windows:

```
./scripts/server.bat
```

Note: `scripts/server.sh` is for Linux and `scripts/server.bat` is for Windows.

## Frontend server

Setup:

```
cd frontend
bun i
```

Run frontend server:

```
bun dev
```
